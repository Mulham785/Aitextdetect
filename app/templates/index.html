How It Works

This tool analyzes text using statistical metrics like perplexity and burstiness to determine the proportion of AI-generated content.

  • Perplexity: Measures text predictability. Lower values indicate more predictable text, common in AI-generated content.
  • Burstiness: Measures variation in sentence structure. Lower values indicate more uniform structure, common in AI-generated content.
  • AI Proportion: Estimated percentage of AI-generated content in the text.
